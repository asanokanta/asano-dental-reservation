# プレビュー

開発サーバーを起動したあと、次のリンクから各画面を開けます。

| 画面 | リンク |
|------|--------|
| トップページ（患者向け） | [http://localhost:3000/](http://localhost:3000/) |
| **医師用 — 予約確認・管理** | [http://localhost:3000/admin](http://localhost:3000/admin) |

医師用画面はログインが必要です。ローカル開発時の初期パスワードは `asano-admin` です（本番は `.env` の `ADMIN_PASSWORD` を使用）。

起動コマンド:

```bash
npm run dev
```

ブラウザ用のリンクページ: [preview.html](./preview.html)（ファイルを開いてからボタンをクリック）
