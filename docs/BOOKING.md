# 自作予約システム

JMDC ではなく、サイト内で予約まで完結します。

## 画面の分け方

| 誰 | URL | できること |
|----|-----|------------|
| **患者** | `/reserve` | 診察券番号 → 日付 → 時間 → 予約確定 |
| **医院** | `/admin` | 患者登録・予約一覧・キャンセル |

## データの保存場所

| ファイル | 内容 |
|----------|------|
| `data/patient_list.csv` | 診察券番号・氏名・ステータス |
| `data/reservations.json` | 予約一覧 |

管理画面から患者を追加すると CSV が更新されます。

## ログイン（医院）

1. http://localhost:3000/admin を開く
2. パスワード（初期値: `asano-admin`）でログイン
3. 本番では `.env` に `ADMIN_PASSWORD=強いパスワード` を設定

## テスト手順

1. `/admin` で患者 `99999` / `テスト太郎` / normal を登録
2. `/reserve` で `99999` を入力
3. 日付・時間を選んで予約
4. `/admin` の予約一覧で確認

## 診療時間（簡易ルール）

- 月火木金: 午前・午後
- 水曜: 午後のみ
- 土曜: 午前のみ
- 日曜: 休診

※ 第2・第4木曜などの細かい休診は未対応。必要なら管理画面で「休診日」機能を追加可能。

## 本番起動

```bash
npm run build
ADMIN_PASSWORD=your-secret npm start
```

`data/` フォルダはサーバー上で書き込み可能にしてください。
